# Conhecendo Objetos no JavaScript

Objetos são coleções dinâmicas de chaves e valores.



Confira um exemplo de objeto:

```javascript
// Objetos literais

const gabriel = {
  nome: `Gabriel Victorino`,
  idade: 25,
  cpf: `525.618.110-69`
};

console.log(gabriel);
/*Output:
{nome: 'Gabriel Victorino', idade: 25, cpf: '525.618.110-69'}
*/
```
